(world state is Survivor)

### Current Features:
- Spear crafting 
  1. rock + rock = spear
  2. cherrybomb + spear = explosive spear
  3. rock + spear = electric spear
  4. (planned) spear + ? = fire spear
- Stat boosts when on mushrooms
